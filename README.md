# patrick204nqh.github.io

Personal website. Built with Jekyll, hosted on GitHub Pages.

[patrick204nqh.github.io](https://patrick204nqh.github.io)

## Local dev

```sh
cd docs
JEKYLL_GITHUB_TOKEN=$(gh auth token) bundle exec jekyll serve
```

Open http://127.0.0.1:4000.

## Deploy

Push to `gh-pages`. GitHub Actions builds and deploys automatically.
